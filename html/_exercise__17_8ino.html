Detailed Description

Demonstration of connecting to WiFi and sending data to ThingSpeak.

This sketch connects the ESP32 to the specified WiFi, then periodically sends two fields of data (RSSI and a digital pin reading) to a ThingSpeak channel. It also demonstrates reading back from the channel.
